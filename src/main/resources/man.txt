Rest Comparison Tool compares two rest services by number of criterias.

To run application provide parameter "file".
    example: java -jar <.jar> --file=<pathToFile>

File should contain:
    url1
    url2
    method
    parameters

Example:
    {
      "url1": "url1",
      "url2": "url2",
      "method": "get",
      "parameters": {
        "parameterName1": "param1Value",
        "parameterName2": "param2Value"
      }
    }
